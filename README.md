# Daniel Palma — Full Stack Developer Portfolio

Personal portfolio showcasing projects, experience, and skills as a Full Stack Developer specializing in modern web technologies.

## Tech Stack

- **Frontend:** React, TypeScript, Tailwind CSS
- **Backend:** C#, Node.js, REST APIs, SQL
- **Tools:** Git, WordPress, Figma

## Features

- Warm, professional design with terracotta accent palette
- Responsive layout across mobile, tablet, and desktop
- Animated sections with Framer Motion
- Consistent vertical rhythm and spacing system
- GitFlow branching workflow

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:5173](http://localhost:5173) to view it in the browser.

## Build

```bash
npm run build
```
